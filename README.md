# 📁 File Sharing Web App

A simple and secure file sharing application built using **React**, **Node.js**, **Express**, and **MongoDB**. This app allows users to upload files and store their metadata securely in a MongoDB database.

## 🔧 Tech Stack

- 🧠 **Frontend**: React
- 🚀 **Backend**: Node.js + Express
- 💾 **Database**: MongoDB Atlas
- 📦 **Middleware**: Multer for handling file uploads
